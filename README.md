This is a real-time current monitoring system for "創鴻高精度智能盤面表 DM4D-An-Rs".

This program supports automatic data saving, charge integral calculation， graphical visualization, and flexible Single/Dual channel operation modes. For more information, please refer to the Tutorial and About in the program.

To run this code, you need to install thses python packages: numpy, matplotlib, pyqt5 and pyserial. To run "pip install numpy matplotlib PyQt5 pyserial" or "conda install numpy matplotlib pyqt5 pyserial"(if you use conda) in Terminal to install these packages.

This program can ONLY run on Linux.
